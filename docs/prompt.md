Create a fully functional 'Space Invaders' game using JavaScript with Canvas.

Architecture Requirements:

1. Modular Design: Separate the code into logic for the Player, Enemies (the grid), Projectiles, and Collision Detection.
2. Game Loop: Implement a standard 60 FPS game loop.
3. Scaling Difficulty: As enemies are destroyed, the remaining ones should move faster.
4. State Management: Include states for 'Start Screen', 'Playing', 'Game Over', and 'Victory'.


Game Mechanics:

- The enemy grid should move horizontally and drop down upon hitting screen edges.
- The player has 3 lives and moves left/right.
- Enemies should fire projectiles at random intervals.
